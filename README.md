# Stata guide

*Stata Guide* is a compact, practical reference guide for working with Stata.  
It is designed to be opened directly inside Stata and used as a quick, searchable companion for everyday data management, analysis, and modeling tasks.

The guide is largely inspired by the **Stata cheat sheets** by **Dr. Tim Essam** and **Dr. Laura Hughes**, and extended with additional examples, notes, and tips learned through hands‑on use over the years.


## Quick start

Open the guide directly in Stata:

```stata
view https://github.com/rampezione/stata-guide/raw/main/guides/index.sthlp
```

The guide opens in Stata’s Viewer, just like a built‑in help file.
